World Wide Web General

1. What are the most essential technologies that made the World Wide Web such a success?

A: The hypertext, hypermedia, HTTP and browsers.

2. How are resources (html pages, images, etc) addressed in the web?

A: Resources are addressed by URL(Uniform Resource Locator), URN (Uniform Resource Names), PURL (Persistent Uniform resource) and URC (Uniform Resource Characteristics). But mostly is URL used.

3. What are the general components of a url?

A: The general components are a scheme, a host, a path, a query string.

Web Server

1. What is a web server

A: A web server is a server that sends documents to clients such as webbrowser.

2. What does a web server?

A: The primary function of a web server is to store, process and deliver web pages to clients.

3. How can content be stored on a web server?

A: Conent can be stored with Hypertext Transfer Protocol (HTTP).

4. How is content in webpages described?

A: It is described with a HTML document.
